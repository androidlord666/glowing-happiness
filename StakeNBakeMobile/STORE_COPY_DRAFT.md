# Store Copy Draft — Solana Mobile Staking

## Short Description
Stake, unstake, consolidate, and send SOL with a premium Solana Mobile-first experience.

## Long Description
Solana Mobile Staking is a focused staking utility for Solana Mobile users.

Built for speed and clarity, it lets you:
- Connect wallet with Solana Mobile Wallet Adapter
- Stake SOL to validator
- Unstake (deactivate) stake accounts
- Consolidate stake accounts with wallet-authority scoped selection
- Send SOL (including `.sol` names)
- Receive SOL with address copy and QR

The app emphasizes clean UX, safe transaction flow, and direct wallet authorization.

### Why users like it
- Mainnet-ready core flows
- Consolidation UX designed to reduce selection mistakes
- Smooth branding and motion tuned for mobile feel
- Configurable explorer and network settings

### Security & Control
- Wallet signs transactions; app does not custody keys
- No seed phrase handling in app UI
- Designed for transparent, user-controlled transactions
